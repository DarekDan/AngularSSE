# Frontend Module

Angular application receiving SSE messages and displaying them.

## Prerequisites
- Node.js 20+
- Angular CLI

## Running Locally
1.  Navigate to `frontend` directory.
2.  Run `npm install`.
3.  Run `ng serve`.
4.  Access at `http://localhost:4200`.

## Docker
1.  Build Image: `docker build -t frontend .`
2.  Run: `docker run -p 80:80 frontend`
